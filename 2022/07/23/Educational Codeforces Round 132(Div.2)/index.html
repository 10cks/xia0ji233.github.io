String Building ### 题目描述 ![1](Educational%20Codeforces%20Round%20132(Div.2)/1.png) ### 题目分析 给你三个门，每个门后可能有🔑，一个钥匙开一个对应的门，问你是否有办法把所有的门打开。很简单，就那到🔑开个门，换新钥匙，看看能开几个门就可以了。 ### 标程 ```c++ #include #define maxn 200005 using namespace std; int a[maxn],b[maxn]; char s[maxn]; void solve(){ int a,b[4]; scanf("%d%d%d%d",&a,&b[1],&b[2],&b[3]); int key=a,cnt=0; while(key){ key=b[key]; cnt++; } if(cnt==3){ puts("YES"); } else{ puts("NO"); } } int main(){ #ifndef ONLINE_JUDGE freopen("1.in","r",stdin); #endif int t=1; cin>>t; while(t--)solve(); return 0; } ``` ## B.
